# Переменные

|  **Переменная**   | **Описание**  | 
| ------------- |:-------------:|
| ```{USER}``` | Реплей: Ссылка |
| ```{PASS}```       | Пароль от VK |
| ```{DATE}```        | Текст - Вложение|
| ```{NONE}``` | Без параметра |
| ```{TEXT}``` | Произвольный текст |
| ```{LOGIN}``` | Логин от VK |
| ```{ENTER}``` | Перенос строки |
| ```{NUMBER}```| Число |
| ```{PARAMS}``` | Параметр |
| ```{COMMAND}``` | Команда |
| ```{CPREFIX}``` | Префикс команд |
| ```{SPREFIX}``` | Префикс скриптов|
| ```{TOKENVKA}``` | Токен VKAdmin |
| ```{TOKENSKR}``` | Токен Sakura |
